The Access Database

This webpage presents the data and analysis created using Microsoft Access as part of my CAT PAT. The database includes:

  • Tables for device information and survey results.
  • Queries with calculated fields.
  • Reports grouped by brand, rating, and features.
  • Forms for data input and navigation.

These elements were used to extract insights on the cost, value, and user satisfaction of smart wearables.

You can access the database here

To heres a quick guide on each of the pages:

There are a total of 12 pages within the Access document. This includes:

Tables:

  • BestWearables: a table showing the low-end, mid-range, and high-end devices from the popular brands.
  • SurveyResults: a table showing all the collected survey data.

Queries:

  • CostVsValue: a table showing the wearables with the best accuracy rating with the lowest average price.
  • RegularUserImpact: a table showing the impact on regular users from the survey.
  • TopFeatureRichDevices: a table showing the highest rated (above 4) devices and with the most features (total features above 10).

Forms

  • Navigation Form: the main form for accessing the data.
  • SurveyQuestionaire: a form to update the survey inputs.
  • SurveyQuestionsList: a form displaying all of the SurveyResults table.
  • WearablesForm: a form to update the BestWearables table.

Reports

  • BrandAverageRating: a report showing a brands average accuracy rating.
  • CostVsValue: a report showing a devices accuracy grouped by brand with each devices price.
  • FeatureRichDevicesReport: a report showing devices that are feature rich and have an accuracy above 4.

    • It is recommended you go through the navigation form first as it will will provide buttons to open, print, or mail each page
