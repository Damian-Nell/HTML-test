About This Project

This project forms part of my Grade 12 IEB CAT task. It is focused on exploring the world of smart wearable technologies. It presents data and analysis gathered during the course of this project using tools such as Microsoft Access, Microsoft Excel, Microsoft Word, and HTML.

The core purpose of the project is to investigate how smart wearables are being used today and to how factors like functionality, cost, user experience, and innovation influence each devices development and adoption The research includes a structured survey, detailed comparisons of popular devices, and visual data analytics.

This site acts as the central hub for accessing all the data and files within this PAT and aims to make the findings accessible, easy to understand, and useful for anyone interested in wearable technology, Whether you're a casual user or enthusiast. It brings together real-world data and emerging trends to highlight both the potential and challenges of wearable technology today.
